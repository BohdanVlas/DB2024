Інструкція з використання CRUD функцій для керування базою даних:

1. Створення нового запису:

   Щоб створити новий запис, скористайтеся відповідною CRUD функцією для потрібної сутності (книги, автора, користувача тощо). Наприклад, для створення нової книги використайте функцію `create_book()` і передайте необхідні параметри:

   SELECT create_book('Назва книги', автор_id, жанр_id, рейтинг, рік_видання);

2. Оновлення існуючого запису:

   Для оновлення інформації про запис скористайтеся відповідною CRUD функцією з параметрами, які потрібно змінити. Наприклад, для оновлення інформації про автора використайте функцію `update_author()`:

   SELECT update_author(ідентифікатор_автора, 'Нове ім'я автора');

3. Видалення запису:

   Щоб видалити запис з бази даних, використайте відповідну CRUD функцію з ідентифікатором запису, який потрібно видалити. Наприклад, для видалення книги використайте функцію `delete_book()`:

   SELECT delete_book(ідентифікатор_книги);

4. Отримання інформації про запис(и):

   Щоб отримати інформацію про запис(и), використайте відповідну CRUD функцію. Наприклад, для отримання інформації про всі книги скористайтеся функцією `get_all_books()`:

   SELECT * FROM get_all_books();

5. Отримання запису за ідентифікатором:

   Для отримання конкретного запису за його ідентифікатором використайте відповідну CRUD функцію з ідентифікатором як параметром. Наприклад, для отримання книги за ідентифікатором використайте функцію `get_book_by_id()`:

   SELECT * FROM get_book_by_id(ідентифікатор_книги);

6. Використання інших CRUD операцій:

   Таким чином, ви можете виконувати інші CRUD операції для керування іншими сутностями бази даних, такими як автори, користувачі, копії книг та інші.
